# Expense-Tracker
An expense tracker built in MERN stack that manages income and expenses and creates bar charts , pie charts, line charts etc. of entered information

## Setup Instructions

### 1. Clone the repository

git clone https://github.com/ShaanSinghWorkSpace/Expense_Tracker.git

### 2. Install dependencies

Frontend:  
cd frontend  
npm install

Backend:  
cd backend  
npm install

### 3. Environment Variables

Create a .env file inside the backend folder and add:

MONGO_URI=your_mongodb_connection_string  
PORT=5000

### 4. Run the application

Start backend:  
cd backend  
npm run dev

Start frontend:  
cd frontend  
npm run dev

## Notes

- Make sure MongoDB Atlas is set up correctly.
- Do not expose your .env file or credentials.
